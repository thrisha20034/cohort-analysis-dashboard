
# Revenue Growth Cohort Analysis Dashboard

An interactive data analytics dashboard simulating a real-world Power BI report, built to showcase end-to-end data engineering and analysis skills.

## 🔗 Live Demo
[View Dashboard](https://thrisha20034.github.io/cohort-analysis-dashboard/)

## 📌 Project Overview
Analyzed 2+ years of customer transaction and engagement data to identify high-value customer segments, retention patterns, and revenue growth opportunities.

## ✨ Features
- Interactive Cohort Retention Matrix with color-coded heatmap
- Revenue Trends with toggleable cohort lines
- Customer Segment drill-down (Champions, Loyal, At Risk, Churned)
- Live CLV Simulator with adjustable parameters
- Quarter-based filtering with real-time KPI updates

## 🛠️ Technologies Used
- SQL & Snowflake — data extraction, transformation, aggregation
- Python (Pandas) — cohort modeling, retention analysis, CLV calculation
- PySpark — large-scale dataset processing and ETL workflows
- Power BI — interactive dashboard design for business stakeholders
- Git & GitHub — version control and project collaboration

## 📊 Key Findings
- High-retention cohorts generated **25–30% higher lifetime revenue**
- Champions segment (24% of customers) drives **61% of total revenue**
- Biggest churn risk window identified at **Month 3 → Month 4**
- Q1 2024 cohorts retained **8pp better** than Q4 cohorts
